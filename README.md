EchoMechanic é uma plataforma de IA desenvolvida para detetar falhas em máquinas industriais, veículos e eletrodomésticos através da análise de frequências sonoras

- Funcionalidades Principais

**Análise de Áudio em Tempo Real**: Upload de ficheiros de som para deteção de falhas (rolamentos, desalinhamento, etc.).
  
**Diagnóstico**: Respostas adaptativas baseadas no perfil do utilizador (Técnico ou Gestor).

**Gestão de Ativos**: Registo e histórico de manutenção para cada máquina.

**Samantha AI Chatbot**: Assistente virtual treinada em normas ISO (ex: ISO 18436) para apoio técnico.

**Relatórios PDF**: Geração automática de relatórios técnicos para equipas de manutenção.

Stack Tecnológica

Backend: FastAPI (Python)

IA: Google Gemini 2.0 Flash API

Base de Dados: SQLite (SQLAlchemy)

Frontend: HTML5, CSS3 (Tailwind/Modern UI), JavaScript

Documentação: FPDF para relatórios técnicos
